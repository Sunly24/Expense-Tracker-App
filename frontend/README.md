# Expense Tracker App

## Getting Started

### Installation

 Clone the repository
```bash
git clone
```

 Navigate to the project directory
```bash
cd expense-tracker/frontend
```

 Install dependencies
```bash
flutter pub get
```

 Navigate to backend directory
```bash
cd expense-tracker/backend
```

 Install dependencies
```bash
npm install
```

 Navigate to backend directory
```bash
cd expense-tracker/backend
```

 Install dependencies
```bash
npm install
```


 Run the application
```bash
flutter run
```
